¿Qué es dominous?

  • Dominous es un videojuego de dominó
  • También es Software libre
  • Utiliza Sistemas Expertos
  • Tutorial, modo laboratorio, sistema de themes, creación de nuevos jugadores...

¿Por qué crear un simulador de dominó?

  • El dominó es un juego complejo
  • No existen dominós libres
  • Reto con IA, interfaces, programación gráfica, audio, diseño gráfico, aplicación multiplataforma...

Calendario

Herramientas

  • Lenguaje de programación — Python
  • Librerías gráficas — SDL + PyGame/Gloss
  • Documentación — Latex, Doxygen/Doxypy
  • Forja RedIRIS — SVN, sistema tickets y bugtracking, foros, listas de correo
  • Otros — PyLint, Photoshop, GIMP, Audacity, vim, make...

Conclusiones 1/2

¿Qué hemos conseguido con este proyecto?

  • Primer simulador libre de dominó
  • Aplicación de todo lo aprendido en la carrera
  • Aprendizaje del juego de dominó
  • Proyecto realizado de forma íntegra

Conclusiones 2/2

Trabajos futuros:

  • Juego en red
  • Servidor web para compartir sistemas expertos entre usuarios
  • Interfaz gráfica para crear nuevos jugadores
  • Analizador de partidas con aprendizaje automático o semiautomático
  • Migración de la aplicación a otros sistemas
